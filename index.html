¿Qué es HTML?

HTML son las siglas de HyperTextMarkupLanguage

A diferencia de un lenguaje de programación o de secuencias de comandos que usa secuencias de comandos para realizar funciones, un lenguaje de marcado usa etiquetas para identificar el contenido

Etiquetas

HTML es un lenguaje que se basa en etiquetas


					
Soy un título

					
Soy una etiqueta

Estructura básica HTML

el documento HTML tiene etiquetas HTML de apertura y cierre.
Todo documento de HTML está rodeado por la etiqueta <html> 


			  <html>
			  ...
			  </html>

Estructura básica HTML

La etiqueta head se usa para el encabezado del documento
En encabezado contiene todos los elementos no visuales que ayudan a que la pagina funcione correctamente. 


			<html>
			<head>
			...					
			</head>
			...
			</html>

Estructura básica HTML

La etiqueta body sigue a la etiqueta de la cabeza
En esta etiqueta se encuentran todos los elementos visuales
Como por ejemplo: títulos, párrafos, listas, imágenes, etc. 


			<html>
			<head>
			...
			</head>
			<body>
			...	
			</body>
			</html>

Estructura básica HTML

La etiqueta title define el título de la pagina


			<html>
				<head>
					<title>RevealJS-Presentation</title>
				</head>
				<body>
				...	
				</body>
			</html>

Estructura básica HTML

La etiqueta meta define la infomación de la información que desplegamos en la pagina web. Son usados para especificar la codificacion de caracteres, descripcion, autor, palabras claves, etc. 


			<html>
				<head>
					<meta charset="UTF-8">
					<meta name="descripcion" content="Reveal JS Framework">	
					<meta name="keywords" content="HTML,CSS,RevealJS">
					<meta name="author" content="Solano Pillaca Jason">
					<title>RevealJS-Presentation</title>
				</head>
				<body>
				...	
				</body>
			</html>

¿Qué es CSS?

CSS significa hoja de estilo de cascada
Sirve para aplicar específicos éstilos a los elementos HTML

¿Cómo utilzizalo?

Hay 3 formas de utilizarlo:
  1. Inline CSS
  2. CSS embebido
  3. CSS externo

Inline CSS

Usando el atributo style en un elemento HTML específico 

            <p style="color:white; background-color:gray;">
                Este es un ejemplo de CSS Inline 
            </p>
Resultado:
Este es un ejemplo de CSS Inline

CSS embebido u interno

Está definido con el elemento html <style>
Se ubica dentro de la etiqueta head 

              <html>
              <head>
              <meta charset="UTF-8">
                <title>Título</title>
                <style>
                  p {
                    color: black;
                    background-color: white;
                  }
                </style>
                </head>
              <body>
              <p>Soy un parrafo</p>
                <p>Soy otro parrafo</p>
                </body>
                </html>
Resultado:

Soy un párrafo

Soy otro párrafo

CSS externo

Con este método el estilo CSS está contenido en un archivo aparte con la extensión .css
Es referenciado en el documento HTML usando el tag link dentro de la etiqueta head 

                            <head>
                            <link rel="stylesheet" href="example.css">
                            </head>
                            <body>
                            <p>Un párrafo</p>
                            <p>Otro párrafo</p>
                            <p>Tercer párrafo</p>
                            </body>

Syntaxis

Las reglas de la syntaxis están compuestas por 3:

  1. Selector: Es a que elemento le aplicamos cierto estilo
  2. Propiedad: Es el estilo u propiedad que le vamos a aplicar
  3. Valor: Es el valor de la propiedad

Despues de asignar un estilo debe de ir el ; 

              h1 {color:orange;}
Selector Propiedad Valor
h1 color orange

Tipos de selectores

Se tiene 3 tipos de selectores:
  1. Selector por tipo
  2. Selector por id y class
  3. Selector descendiente

Selector por tipo

Por ejemplo para estilar todos los párrafos en la página 

                p {
                  color: red;
                  font-size:130%;
               }

Selector por id y class

El selector id te permite estilar un elemento segun su id
El archivo HTML: 

           <div id="intro">
               <p>Este párrafo está dentro de la etiqueta div</p>
           </div>
               <p>Este párrafo está fuera de la etiqueta div</p>
El archivo CSS: 

                #intro {
                  color: violet;
                  background-color: white;
                }

Este párrafo está dentro de la etiqueta div

Este párrafo está fuera de la etiqueta div

Selector descendiente

Este selector se usa para seleccionar ciertos elemtos que son descendientes de otros elementos
Por ejemplo, para seleccionar solo los elementos em dentro de la clase primero que está dentro del id intro
El archivo HTML: 

             <div id="intro">
               <p class="primero">Esto es un<em> párrafo</em></p>
               <p>Esto es un <em> párrafo</em> fuera de la clase.</p>
             </div>
El archivo CSS: 

                #intro .primero em {
                  color: pink; 
                  background-color:gray;
               }

Esto es un párrafo

Esto es un párrafo fuera de la clase

¿Qué es RevealJS?

  • Es un framework opensource de javascript que permite crear presentaciones multimedia impresionantes en la Web usando HTML.
  • Esta dirigido a los desarrolladores, por lo que es necesario saber las bases sobre HTML y CSS.
  • Una gran ventaja frente a otros software de presentaciones, es que al ser un framework es posible agregarle cualquier tecnología.

¿Cómo insertar código?

Insertar código es la característica más llamativa de RevealJS
Para ello solo tenemos que copiar y pegar nuestro código dentro de la etiqueta code, dentro de otra etiqueta pre 

  <div class="reveal">
      <div class="slides">
        <section>
              <pre><code>
              <!--Aqui va el codigo-->
	            </code></pre>
	      </section>
      </div>
  </div>

¿Cómo formatear el código?

Si no le hemos indicado a RevealJS como queremos que se presente el código,
este se vera desordenado y sin ciertas lineas resaltadas
Entonces agregamos los atributos a la etiqueta code
El atributo data-trim se usa para borrar los espacios y así poder espaciarlos a nuestro gusto
El atributo data-line-numbers se usa para enumerar las lineas del código y también resaltar ciertas lineas, si se especifican 

  <div class="reveal">
      <div class="slides">
        <section>
              <pre><code data-trim data-line-numbers="2">
              var myName = "Solano Pillaca Jason"; 
              console.log(`My name is: ${myName}`);
	            </code></pre>
	      </section>
      </div>
  </div>

¿Cómo animar el código?

Para animar el código simplemente creamos 2 diapositivas, el primero con el estado inicial del código y el segundo con el estado final
Cada una de las diaposiitivas debe de tener el atributo data-auto-animate
El código que queremos que se anime dentro de las diapositivas debe de tener el atributo data-id="code-animation" 

    <div class="reveal">
      <div class="slides">
        <section>
          <section data-auto-animate>
            <pre data-id="code-animation"><code data-line-numbers data-trim>
              <!--Estado inicial del código-->
            </code></pre>
          </section>
          <section data-auto-animate>
            <pre data-id="code-animation"><code data-line-numbers data-trim>
              <!--Estado final del código-->
            </code></pre>
          </section>
        </section>
      </div>
    </div>
